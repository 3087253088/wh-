# 智慧财务一体化平台 - 相关功能介绍及代码相关配置

## 一、系统简介

智慧财务一体化平台是一款面向企业内部的财务管理系统，支持费用报销、发票管理、预算控制、审批流程等核心财务功能。系统采用RBAC权限模型，分为管理员、财务人员、普通员工三种角色。

所有后端代码的java版本为jdk21，前端使用的是vue^3.3.0,数据库使用的是mysql，版本选择自己合适的版本即可，文件包含建表语句的sql脚本

## 四、功能说明

### 4.1 登录系统
1. 访问系统首页，进入登录页面
2. 输入用户名和密码
3. 点击"登录"按钮

### 4.2 工作台（首页）
登录后默认进入工作台，根据角色显示不同内容：
- **普通员工**：个人报销总额、我的发票、个人信息
- **财务/管理员**：本月报销总额、待审批单据、预算执行率、员工数量 + 待办事项列表 + 费用趋势图

### 4.3 我的报销
**功能**：提交报销申请、查看报销记录
1. 点击"新增报销"，填写报销信息
2. 保存为草稿或直接提交审批
3. 提交后进入审批流程
4. 可随时查看报销状态（草稿/审批中/已通过/已驳回）

### 4.4 我的发票
**功能**：录入发票、查看发票、发票验真
1. 点击"录入发票"，填写发票信息
2. 系统自动查重，防止重复录入
3. 可对发票进行验真（需要调用第三方接口，暂时不支持）

### 4.5 我的审批（财务/管理员）
**功能**：审批待处理的报销单
1. 查看待审批列表
2. 点击"通过"或"驳回"
3. 填写审批意见
4. 审批完成后申请人会收到通知

### 4.6 预算管理（财务专用）
**功能**：编制预算、调整预算、查看预算执行情况
1. 按部门+费用类型+月份编制预算
2. 实时查看预算使用进度
3. 超预算预警

### 4.7 发票管理（财务专用）
**功能**：查看全公司发票、批量验真

1. 查看所有员工的发票
2. 支持按条件筛选
3. 支持批量验真（需要调用第三方接口，暂时不支持）

### 4.8 员工信息（财务专用）
**功能**：管理员工信息
1. 查看员工列表
2. 新增/编辑员工
3. 员工离职/复职操作

### 4.9 人员管理（管理员专用）
**功能**：系统用户管理
1. 查看所有用户
2. 新增/编辑用户
3. 分配角色
4. 重置密码
5. 启用/禁用账号

### 4.10 我的信息
**功能**：查看和修改个人信息、修改密码、上传头像

## 五、常见问题

### Q1：登录失败？
- 检查用户名密码是否正确
- 确认账号未被禁用
- 检查后端服务是否正常运行

### Q2：提交报销后找不到单据？
- 查看状态筛选是否为"全部"
- 草稿状态的单据需要点击"提交"才会进入审批

### Q3：关于发票验真功能
- 确认发票号码、开票方、金额等信息是否正确
- 第三方验真接口需要调用第三方接口，暂时不支持
